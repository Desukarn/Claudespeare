## Next Steps

After reviewing this report:

1. **Review flagged issues** - Read each finding and determine if error or intentional
2. **Mark intentional changes:**
   - Update CHARACTERS.md if character details legitimately evolved
   - Add note in ARCS.md explaining character growth arcs
   - Update WORLD.md if world rules expanded/clarified
3. **Fix genuine errors:**
   - Revise chapter prose to fix contradictions
   - Update chapter files and save
   - Ensure consistency with established baseline
4. **Re-run consistency check:**
   - Execute `/write:check` again after fixes
   - Verify issues resolved
   - New report should show reduced flag count
5. **Document resolution:**
   - Use Notes section below to track what you fixed
   - Mark items as "fixed", "intentional", or "ignore"
