<task type="auto">
  <name>Task 1: Create YOLO expansion workflow</name>
  <files>.claude/skills/write/yolo-expand.md</files>
  <action>
Create the YOLO mode expansion workflow that takes vague ideas and generates structured story elements.

Workflow structure:

1. **Entry validation**: Check PROJECT.md frontmatter has `mode: yolo`, error if In-Depth mode

2. **Gather vague idea**:
   - If premise already in PROJECT.md → use it
   - If premise is very short (< 20 words) → ask writer to elaborate slightly
   - Examples to show: "detective with magic powers", "space station mystery", "fantasy rebellion story"

3. **Premise expansion** (YOLO-01):
   - Analyze vague idea for core elements: protagonist type, conflict type, setting type
   - Generate 2-3 sentence expanded premise with:
     - Protagonist (who)
     - Core conflict (what's at stake)
     - Setting (where/when)
     - Unique hook (what makes it interesting)
   - Example expansion: "detective with magic powers" → "A homicide detective who secretly possesses telepathy must solve murders in a city where magic is illegal, while hiding her abilities from her mundane partner and avoiding the Inquisition hunting magic users."

4. **Character sketch generation** (YOLO-02):
   - From expanded premise, identify 2-4 key characters:
     - Protagonist (required)
     - Antagonist or opposing force (required)
     - 1-2 supporting characters (optional based on premise)
   - For each character generate:
     - Name (suggest appropriate to genre)
     - Role (protagonist/antagonist/ally/rival)
     - 3 core personality traits
     - Brief background (2-3 sentences)
     - Primary goal
   - Write to `stories/{slug}/CHARACTERS.md` using yolo-characters.md template

5. **Plot structure suggestion** (YOLO-03):
   - Generate simple 3-act structure:
     - Act 1: Setup - inciting incident idea
     - Act 2: Confrontation - 2-3 major complications
     - Act 3: Resolution - climax concept
   - Keep it high-level (bullets, not detailed scenes)
   - Write to `stories/{slug}/OUTLINE.md` using yolo-outline.md template

6. **Skip-to-writing enablement** (YOLO-04):
   - Update PROJECT.md status to "ready-to-write"
   - Explain: "YOLO setup complete. You can now start writing chapters. Details will emerge during drafting."
   - Note: "You can add detail to CHARACTERS.md and OUTLINE.md anytime, or let the story evolve organically."

Implementation notes:
- Use Claude's analysis to expand premise intelligently (not just template filling)
- Character names should fit genre (fantasy → Aldric, sci-fi → Nova Chen, contemporary → Sarah Martinez)
- Keep tone encouraging - YOLO is about momentum, not perfection
- No world-building or character arcs files in YOLO mode (optional for writer to add later)

Create simplified templates for YOLO mode that provide structure without overwhelming detail.

**File: .planning/templates/yolo-characters.md**

Structure (LENGTH-01 - lighter than full template):
- Title: "Characters: {STORY_TITLE}"
- Section: Main Characters
  - For each character (2-4 total):
    - Name
    - Role (protagonist/antagonist/ally/rival)
    - Core traits (3 bullet points)
    - Brief background (2-3 sentences)
    - Primary goal (1 sentence)
    - Voice note (optional - 1 sentence about how they speak)
- Section: Quick Reference
  - Bullet list of all character names with 1-word role
- Note at bottom: "This is a YOLO starter template. Add details as your story develops, or keep it simple and discover characters through writing."

Total length: ~60 lines (vs full template's 117 lines)

**File: .planning/templates/yolo-outline.md**

Structure (LENGTH-01 - lighter than full template):
- Title: "Plot Outline: {STORY_TITLE}"
- Section: Three-Act Structure
  - Act 1 (Setup):
    - Inciting incident (2-3 sentences)
    - Commitment point (1-2 sentences)
  - Act 2 (Confrontation):
    - Major complications (3-5 bullets of key events)
    - Midpoint twist (2-3 sentences)
  - Act 3 (Resolution):
    - Climax concept (2-3 sentences)
    - Resolution (1-2 sentences)
- Section: Plot Threads to Track
  - Checklist of 3-5 key storylines to remember
- Note at bottom: "This is a YOLO starter outline. Major beats are here; scenes will emerge during writing. Adjust as story develops."

Total length: ~70 lines (vs full template's 128 lines)

Implementation notes:
- Both templates use same markdown structure as full templates for consistency
- Placeholders use {VARIABLE} syntax for easy replacement
- Comments guide writers on what to fill in
- Emphasize flexibility: "starter", "will emerge", "adjust as needed"
