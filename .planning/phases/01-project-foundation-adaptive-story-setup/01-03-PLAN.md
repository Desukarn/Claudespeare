  <name>Task 1: Create In-Depth planning workflow</name>
  <files>.claude/skills/write/indepth-plan.md</files>
  <action>
Create the In-Depth mode planning workflow that guides writers through comprehensive story development.

Workflow structure:

1. **Entry validation**: Check PROJECT.md frontmatter has `mode: indepth`, error if YOLO mode

2. **Planning phase overview**:
   - Explain the In-Depth workflow: "We'll create detailed planning documents for characters, plot, world, arcs, and themes"
   - Show estimated time: "This process takes 30-60 minutes but builds a solid foundation"
   - Emphasize control: "You drive the planning - I'll prompt and structure your ideas"

3. **Character development** (STORY-01):
   - Prompt for number of main characters (typically 2-4)
   - For each character, prompt for:
     - Name
     - Role (protagonist/antagonist/supporting)
     - Age and occupation
     - 3-5 personality traits (with examples)
     - Background (upbringing, formative events)
     - External goal (what they want in story)
     - Internal goal (what they need to learn/overcome)
     - Conflicts (internal and external)
     - Voice/speech patterns
     - Physical description (key traits)
   - Generate CHARACTERS.md from full template with all details
   - For length-aware sections: Include advanced sections (relationships map, consistency notes) for novel-length, skip for short stories

4. **Plot structure** (STORY-02):
   - Guide through three-act structure:
     - Act 1: Opening scene, inciting incident, first plot point
     - Act 2: Rising action, midpoint, complications, second plot point
     - Act 3: Climax, resolution, denouement
   - Prompt for 2-4 subplots if novel-length
   - For each act, ask for 3-5 key scenes
   - Generate OUTLINE.md from full template
   - Length-aware: Novel gets detailed scene list, short story gets just major beats

5. **Character arcs** (STORY-03):
   - For each main character:
     - Starting beliefs and flaws
     - Arc type (positive change, flat, negative, corruption)
     - 3-5 key growth moments (what happens, what they learn, how they change)
     - Ending beliefs and growth achieved
   - Generate ARCS.md from full template
   - Length-aware: Novels track full arc with 5+ growth moments, short stories track simplified arc with 3 moments

6. **World-building** (STORY-04):
   - Setting details (time, location, scope)
   - World rules (magic/tech system, limitations, costs) if genre requires
   - Social structure (governance, classes, institutions)
   - Culture (values, taboos, customs, religion)
   - Geography (2-4 key locations with descriptions)
   - History timeline (3-6 major events)
   - Generate WORLD.md from full template
   - Length-aware: Novels get comprehensive world-building, short stories get basic setting only

7. **Themes and motifs** (STORY-05):
   - Update PROJECT.md with:
     - 3-5 core themes
     - Story inspiration notes
     - Target audience
     - Voice and style decisions (POV, tense, tone)
     - Planned word count
   - These were partially captured in init, now expand with planning insight

8. **Planning completion**:
   - Update PROJECT.md status to "planning-complete"
   - Summary of what was created
   - Next step: "Ready to begin drafting - use /write:chapter when ready"

Implementation notes:
- Use conversational prompting, not forms
- Show examples for each question to guide writer
- Allow skipping optional sections
- Read PROJECT.md `length` field to determine template depth
- Length mapping: short-story → minimal sections, novella → moderate sections, novel → full sections
