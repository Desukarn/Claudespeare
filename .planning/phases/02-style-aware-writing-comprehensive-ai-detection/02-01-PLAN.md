<task type="auto">
  <name>Task 1: Create style analysis workflow</name>
  <files>.claude/skills/write/style-seed.md</files>
  <action>
Create `/write:style` command workflow that:

1. Validates story project exists (reads PROJECT.md from stories/{slug}/)
2. Prompts writer for 2-3 sample paragraphs of desired writing style
3. Analyzes samples for quantifiable patterns:
   - Average sentence length (word count per sentence)
   - Sentence length variety (mix of short/medium/long)
   - Vocabulary level (simple/moderate/literary)
   - Dialogue vs description ratio
   - Tone markers (formal/casual, lyrical/direct, descriptive/sparse)
   - Paragraph structure (length, pacing)

4. Prompts writer to define voice characteristics:
   - Point of view (first-person, third-limited, third-omniscient)
   - Tense (past, present)
   - Formality level (casual, moderate, formal)
   - Narrative distance (close, moderate, distant)

5. Creates stories/{slug}/STYLE.md from style-profile.md template
6. Saves analysis results and voice config in STYLE.md
7. Confirms completion and explains that style will be referenced during chapter generation

Include helpful examples and explanations. Conversational tone similar to indepth-plan.md. Estimated time: 10-15 minutes.
  </action>
