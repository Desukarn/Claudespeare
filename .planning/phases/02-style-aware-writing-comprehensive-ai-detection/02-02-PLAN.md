  <name>Task 1: Create chapter generation workflow</name>
  <files>.claude/skills/write/write-chapter.md</files>
  <action>
Create `/write:chapter` command workflow that:

1. Validates story project exists (reads PROJECT.md)
2. Loads story context:
   - Read STYLE.md (if exists) for generation guidance
   - Read CHARACTERS.md or yolo-characters.md for character voices
   - Read OUTLINE.md or yolo-outline.md for plot structure
   - Read ARCS.md (if exists) for character development tracking
   - Read WORLD.md (if exists) for setting consistency
   - Read existing chapters from chapters/ directory for continuity

3. Prompts writer for chapter information:
   - Chapter number (auto-suggests next number)
   - Chapter focus (which plot beat or scene from outline)
   - POV character (from CHARACTERS list)
   - Scene breakdown (optional - writer can specify scenes or let system suggest)

4. Context-aware generation instructions:
   - Reference STYLE.md sentence patterns, vocabulary, tone
   - Match character voice from CHARACTERS.md for POV and dialogue
   - Track plot threads mentioned in OUTLINE.md
   - Adapt output length based on PROJECT.md story length:
     * Short story: 500-1000 words per chapter, concise
     * Novella: 1000-2500 words per chapter, moderate detail
     * Novel: 2000-4000 words per chapter, rich descriptions
   - Maintain world consistency from WORLD.md
   - Continue character arc from ARCS.md

5. Generate prose scene-by-scene (if multiple scenes):
   - Each scene as separate generation with context
   - Allow writer to review/edit between scenes
   - Confirm before moving to next scene

6. Save chapter to stories/{slug}/chapters/chapter-{NN}.md using chapter.md template
7. Track chapter in PROJECT.md Progress section
8. Suggest next chapter focus based on OUTLINE.md

Include detailed examples of good vs generic prose. Conversational prompting style. Estimated time: 20-40 minutes per chapter depending on length.

CRITICAL: Always reference STYLE.md patterns explicitly in generation instructions. Don't just "consider" style - actively match sentence length, vocabulary level, and tone from analysis.
