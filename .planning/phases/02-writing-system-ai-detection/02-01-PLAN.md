<task type="auto">
  <name>Task 1: Create chapter generation template</name>
  <files>.planning/templates/chapter.md</files>
  <action>
Create a comprehensive chapter generation template (.planning/templates/chapter.md) with the following structure:

**Metadata section:**
- Chapter number
- Working title
- POV character
- Timeline/setting
- Status (draft/revision/final)
- Word count target

**Context References section:**
- @stories/{STORY_NAME}/PROJECT.md — genre, premise, themes
- @stories/{STORY_NAME}/CHARACTERS.md — character profiles for consistency
- @stories/{STORY_NAME}/OUTLINE.md — current position in plot structure
- @stories/{STORY_NAME}/ARCS.md — character transformation stage
- @stories/{STORY_NAME}/WORLD.md — setting details and rules

**Chapter Goals:**
- Plot objectives (what happens in this chapter)
- Character development (how characters grow/change)
- Arc progression (which arc moments occur here)
- Plot threads to advance (specific threads from outline)
- Plot threads to track (ensure continuity)

**Scene Breakdown:**
- List 3-5 scenes with purpose and key beats
- Scene reference: @stories/{STORY_NAME}/chapters/{CHAPTER_NUMBER}/scenes/{SCENE_NUMBER}.md

**Voice Consistency Checklist:**
For each POV character, include reminders:
- Speech patterns from CHARACTERS.md
- Personality traits to reflect
- Current emotional state from arc progression

**Prompt for Generation:**
Structured prompt template that:
- References all context files
- States chapter goals clearly
- Specifies voice requirements
- Includes plot thread tracking
- Requests consistency with established facts

**Post-Generation Checklist:**
- [ ] All context files referenced correctly
- [ ] Character voices match CHARACTERS.md patterns
- [ ] Plot threads from outline advanced as planned
- [ ] Arc progression aligns with ARCS.md stage
- [ ] World rules from WORLD.md respected
- [ ] Scene breakdown complete with links

Use {PLACEHOLDER} syntax throughout for easy instantiation.
Minimum 80 lines with comprehensive guidance and examples.
