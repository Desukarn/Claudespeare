  <name>Task 1: Create comprehensive initialization workflow</name>
  <files>.claude/skills/write/new-project.md</files>
  <action>
Create `/write:new-project` workflow that implements GSD-style comprehensive initialization with questioning flow and option presentation.

**Questioning Flow:**
1. Ask for story title (validate uniqueness by checking stories/ directory)
2. Present story length options with clear descriptions:
   - Short Story (1k-7.5k words): Quick, focused narrative
   - Novella (7.5k-40k words): Extended single plotline
   - Novel (40k-100k+ words): Full-length with subplots
3. Present workflow mode options:
   - YOLO Mode: Start from vague idea, minimal planning, discovery writing
   - In-Depth Mode: Comprehensive planning before drafting
4. Ask for genre with common examples (Fantasy, Sci-Fi, Mystery, Romance, Literary Fiction, Thriller, etc.)
5. Ask for premise/logline (1-2 sentences) with examples:
   - Good: "A detective haunted by her past must solve a murder that mirrors her own trauma"
   - Bad: "A story about a detective"
6. Optionally ask if writer wants to set word count goal now or later
7. Optionally ask if writer wants to define themes now or later

**Option Presentation:**
- Format all options clearly with numbers or letters
- Show brief description for each option
- Indicate what comes next after each choice
- Use formatting: **bold for options**, bullets for lists, clear whitespace

**Project Creation:**
1. Create stories/{title-slug}/ directory structure
2. Generate PROJECT.md from story-project.md template with all metadata populated
3. Create placeholder files based on story length and mode:
   - YOLO mode: Create yolo-characters.md and yolo-outline.md placeholders
   - In-Depth mode: Create characters.md, outline.md, arcs.md, world.md placeholders
4. Display completion message showing what was created
5. Provide clear next steps based on mode:
   - YOLO: "Run /write:expand to transform your vague idea into a workable premise"
   - In-Depth: "Run /write:plan to begin detailed story planning"

**Validation:**
- Check for existing story with same title slug
- Require all mandatory fields (title, length, mode, genre, premise)
- Validate premise is at least 10 words
- Gracefully handle optional fields (goals, themes) with "skip for now" option

**Error Handling:**
- Clear error messages for validation failures
- Suggest corrections (e.g., "Title 'my-story' already exists, try 'my-story-v2'")
- Allow retry without losing previous answers

**Estimated Duration:** 5-10 minutes
