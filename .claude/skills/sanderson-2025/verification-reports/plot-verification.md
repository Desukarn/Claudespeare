**Source Quote** (Lecture #3): "He employs the 'Yes, But / No, And' technique:
- Character attempts solving a problem (Yes), encountering complications

**Why It Matters**: This is a FUNDAMENTAL plot escalation technique that directly supports the "Progress" element of the framework.

**Where to Add**: Section 2 (Progress Tracking), subsection on "Creating Escalation"

**Suggested Addition**:
```markdown
#### Creating Escalation with Yes, But / No, And

Sanderson's escalation technique ensures each scene raises stakes:

**Yes, But**:
- Character achieves their goal (Yes)
- BUT complications arise that create new problems

**No, And**:
- Character fails to achieve their goal (No)
- AND circumstances worsen beyond the initial failure

**Example**:
- Hero tries to steal the artifact (Yes, they get it)
- BUT the theft triggers an alarm (complication)
- Guards surround them (No, they don't escape)
- AND the artifact is cursed (circumstances worsen)

**Application**: Each scene should move forward while simultaneously raising stakes.
```

#### The Try-Fail Cycle (Discovery Writers)

Sanderson notes that "nearly all discovery writers employ variations of this cycle" as a natural progression structure:

**Cycle Structure**:
1. **First Attempt**: Character tries obvious solution → Fails
   - Reveals: Character underestimated the problem
2. **Second Attempt**: Character tries more sophisticated approach → Fails
   - Reveals: Internal flaws or deeper complications
3. **Third Attempt**: Character succeeds after growth/new knowledge
   - Reveals: Character has truly transformed

**Function**: Each failure provides new information, raises stakes, and demonstrates character/antagonist capabilities.

### Hanging a Lantern (Plot Hole Management)

When your story contains unexplained elements that might break reader immersion, acknowledge them subtly through character perspective.

**Technique**:
- Character notices the oddity
- Character acknowledges it's strange
- Character (and reader) moves on, trusting answers will come

**Example**: "Why is there a gorilla in that phone booth? I'll deal with that later."

**Function**: Signals authorial control and intentionality without hand-holding.

**Warning**: Use sparingly. Overuse draws excessive attention to problems.

### Story Circle Framework (Alternative Structure)

Sanderson prefers Dan Harmon's Story Circle to Joseph Campbell's Hero's Journey as "more flexible."

**The Circle**:
1. **Departure**: Character leaves comfort zone
2. **Escalating Trials**: Character faces challenges forcing growth
3. **Crisis**: Greatest challenge/lowest point
4. **Transformation**: Character gains necessary knowledge/power

**Application**: Can be used for entire story or individual character arcs within larger narrative.
