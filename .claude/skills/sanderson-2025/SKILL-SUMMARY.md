# Brandon Sanderson 2025 Writing Skills - Summary

## Overview
This collection contains comprehensive writing skills based on Brandon Sanderson's 2025 BYU Creative Writing lectures. Each skill is designed for integration with Claudespeare and other writing tools.

**Key Concepts**:
- Major Dramatic Question

**Source**: Lecture #7

## How These Skills Work Together

### The Sanderson Holistic Approach

**Sanderson's Core Philosophy**: "A novel thrives when the author develops strong ideas across all three pillars—world, characters, and plot—simultaneously rather than perfecting one before starting another."

### Integration Workflow

**1. Start with Core Concept**:
- Use **Worldbuilding Tools** to establish basic setting
- Apply **Promise-Progress-Payoff** to identify your Major Dramatic Question
- Use **Character Framework** to create protagonist who fits world and drives plot

**2. Develop Iteratively**:
- **Character** needs drive **plot** needs
- **Plot** needs reveal **worldbuilding** needs
- **World** constraints shape **character** development
- Each informs and improves the others

**3. Deepen Systematically**:
- Apply **Magic Laws** (if applicable) to make magic integral to story
- Use **Character Framework** to ensure cast is balanced and distinct
- Apply **Promise-Progress-Payoff** to structure narrative momentum
- Use **Worldbuilding Tools** to ensure coherence and consistency

**4. Revise with Purpose**:
- **Character Pass**: Use Proactive-Relatable-Capable checklist
- **Plot Pass**: Verify Promise-Progress-Payoff framework
- **Magic Pass**: Check Laws compliance and consistency
- **World Pass**: Ensure coherence and sensory grounding

### Example: Building a Story Using All Skills

**Concept**: "A healer discovers their magic is draining life from others"

**Step 1: Worldbuilding** (Worldbuilding Tools)
- Establish magic system rules (Sanderson's Laws)
- Cultural attitudes toward healing
- Economic/social impact of healing magic
- Physical setting where story occurs

**Step 2: Character** (Proactive-Relatable-Capable)
- **Proactivity**: Healer actively seeks to help people (drives plot)
- **Relatability**: Horrified by discovery, self-aware, trying to find solution
- **Capability**: Skilled healer, but limitation creates vulnerability
- **Motivation**: Wants to help without harm
- **Arc**: Grows from reactive horror to proactive solution-seeking

**Step 3: Plot** (Promise-Progress-Payoff)
- **Promise**: Will healer find way to heal without harming?
- **Progress**: Each healing attempt reveals more about how magic works
- **Payoff**: Discovers magic's true source, makes choice about using it
- **Signposts**: Clues about magic, relationship developments, moral dilemmas

**Step 4: Magic Integration** (Sanderson's Laws)
- **Law One**: Define how healing works (hard magic for problem-solving)
- **Law Two**: Life-drain is limitation creating conflict
- **Law Three**: Explore implications deeply rather than adding more powers

**Result**: Coherent story where world, character, plot, and magic system reinforce each other.

## Common Patterns Across All Skills

### The "Depth Over Breadth" Philosophy
- Few elements deeply explored > Many shallow elements
- Appears in: Magic (Law 3), Worldbuilding (core principle), Character (motivation depth), Plot (Major Dramatic Question focus)

### The "Constraints Create Interest" Principle
- Limitations more interesting than abilities
- Appears in: Magic (Law 2), Character (flaws > perfection), Plot (obstacles create progress), Worldbuilding (scarcity creates drama)

### The "Show Don't Explain" Technique
- Reveal through action, not exposition
- Appears in: All skills emphasize integration over info dumps
- Worldbuilding: Pyramid of Abstraction
- Character: Voice through dialogue
- Plot: Promises through opening action
- Magic: Demonstrate through use

### The "Service to Story" Priority
- Every element must serve narrative
- Appears in: Worldbuilding (core philosophy), Magic (solve problems only if understood), Character (motivation drives action), Plot (payoff serves promise)

### The "Consistency is Crucial" Standard
- Internal logic must be maintained
- Appears in: All skills include consistency checklists
- Required for reader trust
- Foundation of immersion

## Troubleshooting Guide

### If Story Feels Flat:
- Check **Character Framework**: Are pillars present and balanced?
- Check **Promise-Progress-Payoff**: Are there enough signposts?
- Check **Worldbuilding**: Is world sensory and present?

### If Plot Feels Aimless:
- Apply **Promise-Progress-Payoff**: Define Major Dramatic Question
- Check **Character Framework**: Does protagonist have clear motivation?
- Verify promises are established early

### If Characters Feel Wooden:
- Apply **Character Framework**: Check motivation (60% of problems are here)
- Verify pillars are present
- Check voice distinction in dialogue

### If World Feels Generic:
- Apply **Worldbuilding Tools**: Go deeper on existing elements
- Use Pyramid of Abstraction for sensory detail
- Check for Earth-clone syndrome

### If Magic Feels Inconsistent:
- Apply **Sanderson's Laws**: Document actual rules used
- Check consistency of limitations
- Verify reader understanding matches problem-solving use

### If Ending Feels Unsatisfying:
- Apply **Promise-Progress-Payoff**: Does payoff match promise?
- Check if resolution uses only established elements
- Verify surprise + inevitable balance
