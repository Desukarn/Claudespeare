# Sanderson 2025 Writing Skills for Claudespeare

## Overview

This collection contains comprehensive writing skills based on Brandon Sanderson's 2025 BYU Creative Writing lecture series. These skills are designed to enhance Claudespeare's story creation workflow with proven techniques from one of fantasy's most successful authors.

## What's Included

### Core Skills (4 Complete Frameworks)

1. **promise-progress-payoff.md** - Plot structure framework
2. **proactive-relatable-capable.md** - Character development framework
3. **sanderson-laws-of-magic.md** - Magic system design framework
4. **worldbuilding-tools.md** - World creation framework

### Supporting Documents

- **SKILL-SUMMARY.md** - Overview and integration guide
- **TEST-APPLICATION.md** - Verified test case showing skills in action

## Quick Start

### For New Stories

When using Claudespeare's `/new-project` command:

1. **Start with worldbuilding-tools.md** to establish your setting
2. **Apply sanderson-laws-of-magic.md** (if using magic)
3. **Use proactive-relatable-capable.md** to create your cast
4. **Structure plot with promise-progress-payoff.md**

### For Revision

When using Claudespeare's `/revise` command:

1. **Character pass**: Apply proactive-relatable-capable checklist
2. **Plot pass**: Verify promise-progress-payoff framework
3. **Magic pass**: Check sanderson-laws compliance
4. **World pass**: Ensure coherence using worldbuilding-tools

## Key Principles Across All Skills

1. **Depth Over Breadth** - Explore few elements deeply vs. many shallowly
2. **Constraints Create Interest** - Limitations more engaging than abilities
3. **Show Don't Explain** - Reveal through action, not exposition
4. **Service to Story** - Every element must serve narrative
5. **Consistency is Crucial** - Internal logic must be maintained

## Quick Reference Guide

**Need help with...**
- **PLOT?** → promise-progress-payoff.md
- **CHARACTERS?** → proactive-relatable-capable.md
- **MAGIC?** → sanderson-laws-of-magic.md
- **WORLD?** → worldbuilding-tools.md

**One-Sentence Summaries**:
- **Promise-Progress-Payoff**: Set expectations, show meaningful advancement, deliver surprising inevitability
- **Proactive-Relatable-Capable**: Excel in one pillar, grow in another, maintain third for consistency
- **Sanderson's Laws**: Understand = solve, limitations > powers, expand > add
- **Worldbuilding**: Serve story, go deep not dense, sense over explanation
