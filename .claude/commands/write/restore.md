---
description: Restore a previous chapter version (current version saved automatically)
---

Restore a chapter to a previous version. Current version is saved as a new revision before restoring.

Read and follow the `/write:restore` section in `.claude/skills/write/chapter-versions.md`.

Arguments: $ARGUMENTS

Expected format: `<chapter-name> <version-number>` (e.g., `chapter-1 2` or `1 v02`)

If arguments incomplete, ask the user for chapter name and version number.
